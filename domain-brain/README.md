# Domain Brain

Distilled project knowledge for AI and human contributors.

This repository is a local-first text-to-speech monorepo. The domain-brain keeps the core terms, invariants, flows, and state machines short and explicit so future changes stay aligned with the actual runtime behavior.

## Contents

- `glossary.md`: canonical domain terms
- `invariants.md`: rules that must not be violated
- `edge-cases.md`: important exceptional scenarios
- `entities/`: per-entity distilled notes
- `flows/`: core flow summaries
- `state-machines/`: structured state transition definitions

Keep these files short, precise, and updated alongside code changes.
